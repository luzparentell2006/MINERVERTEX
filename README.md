# 🚀 MINERVERTEX - Plataforma de Inversión en Criptomonedas

## 📱 Descripción
MINERVERTEX es una plataforma moderna de inversión en criptomonedas con sistema de referidos, minería diaria y gestión segura de fondos mediante integración con Binance API.

## ✨ Características Principales

### 🔐 Autenticación & Registro
- ✅ Registro responsivo con validación completa
- ✅ Campos: Nombres, Apellidos, Documento, Correo, Teléfono
- ✅ Sistema de login seguro con JWT

### 💰 Dashboard (Hogar)
- ✅ Visualización de saldo en USDT
- ✅ 6 opciones circulares principales
- ✅ Carrusel de imágenes publicitarias

### 📊 Sistema de Inversiones
- ✅ Inversión: 50 USDT por 31 días
- ✅ Ganancia: 50 USDT + comisiones diarias
- ✅ Minería diaria aleatoria
- ✅ Comisión referidos: 5%

## 🛠️ Stack Tecnológico
- Frontend: React 18, TailwindCSS
- Backend: Node.js, Express, MongoDB
- API: Binance API, JWT

## 🚀 Inicio Rápido

```bash
git clone https://github.com/luzparentell2006/minervertex.git
cd minervertex
npm run install-all
npm run dev
```

Frontend: http://localhost:3000
Backend: http://localhost:5000
